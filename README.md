# Zombie FrostLands 

## Descripción
Simulador de supervivencia zombie en tercera persona desarrollado en Unity. Sobrevive en un entorno post-apocalíptico, completa misiones, gestiona tu energía y arsenal contra hordas de zombies.

**Mecánicas del Juego**
    Sistema de Energía
La energía disminuye al realizar acciones (correr, disparar, atacar)

Recuperar energía quedándose quieto o consumiendo comida

Gestión estratégica de la energía es crucial para la supervivencia

    Sistema de Comida
Comida spawn aleatoriamente en zonas de zombies

Comida segura → Restaura energía

Comida envenenada → Reduce energía (riesgo aleatorio)

Toma decisiones cuidadosas al consumir recursos encontrados

    Comportamiento de NPCs
NPCs inteligentes que detectan zombies cercanos

Comportamiento de huida cuando detectan amenazas

Interacción realista con el entorno de juego

    Misiones Principales
Limpieza Inicial - Elimina 10 zombies

Caza Mayor - Elimina 25 zombies

Supervivencia Extrema - Sobrevive 3 minutos

    Estrategias de Supervivencia
Gestiona tu energía - No corras innecesariamente

Busca comida con cuidado - Evalúa el riesgo de envenenamiento

Protege a los NPCs - Ellos te darán misiones importantes

Entra en zonas de misión para activar objetivos

Recoge armas del suelo para expandir tu arsenal
